# Repository-to-Resume Package Minimal Prompt Template

生成最小但完整的 PM 求职资料包。目标仓库只读，不修改原项目 README、源码、docs 或部署配置。

## Required files

必须包含 `12_OWNERSHIP_AND_SOURCE_APPENDIX.md`，即 Ownership and Source Appendix。

如果用户没有提供 ownership 信息，生成待填写模板；如果用户已提供 ownership 信息，整理为可引用证据附录。

## User confirmation reminder

最后提醒用户补充：

1. GitHub 账号是否本人所有；
2. 哪些内容本人主导；
3. 哪些内容 AI/coding 工具辅助；
4. 哪些内容不是本人完成；
5. 哪些 claim 不能写进简历。

# Action Card Format

The round output must always include:

- Round judgment
- Diagnostic snapshot
- Top three priorities
- Active action card
- Integrity note

Expand only one active action card.

## Active action card

Include:

- objective
- why now
- concrete steps
- method or template
- acceptance criteria
- artifact to return for verification
- supported job requirement
- estimated effort
- reduced-scope version for limited time
